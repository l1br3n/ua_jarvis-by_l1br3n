# 🤖 UA Jarvis (by l1br3n)


Персональний голосовий асистент з українським інтерфейсом, побудований на базі Vosk для розпізнавання мови та інтегрований зі штучним інтелектом Google Gemini.


## ✨ Що нового в цій версії
* **Перехід на модульну систему**: Тепер всі файли згруповані по папках, та Джарвіс тепер може працювати навіть тоді, коли буде помилка в якомусь модулі.

* **Переписані функціїї**: Заради модульності я приняв рішення повністю переписати структуру та логіку окремих файлів.

---


## 🚀 Основні можливості
* **Керування ПК**: Закриття та згортання вікон, вставка тексту з буфера обміну, керування гучністю та мікрофоном.

* **Мультимедіа та Spotify**: Запуск персональних плейлистів, перемикання треків та пошук пісень через Spotify API (Див. [короткий відео-туторіал як їх отримати](https://youtu.be/0fhkkkRuUxw?si=U74GcDt8seGxgGlW)) також потрібно добавити озвучку на конкретний плейлист в файлі `media_control.py`.

* **Запуск програм та ігор**: Відкриття софту через налаштовані псевдоніми (аліаси) та запуск ігор у Steam.

* **Інтернет-пошук**: Швидкий пошук інформації в Google, відкриття сайтів та перегляд відео на YouTube за голосовим запитом.

* **Комунікація**: Надсилання повідомлень у Telegram контактам із вашої бази та підключення до голосових каналів Discord.(також потрібно добавити озвучку на конкретний канал в файлі `communication.py`.)

* **Робота з клавіатурою**: Вводити текст який він почує та використовувати гарячі клавіші.

* **Можливість спілкування з потужною LLM**: Інтегрований Gemini-2.5-flash, який відповідає в стилі Джарвіса з фільму (Працює не кожен раз, скоріше за все через те що може неправильно розпізнати запит).

* **Для розробників**: Швидке відкриття робочих проєктів у Visual Studio Code (в цій версії можливі помилки).

## ⚙️ Налаштування

### 1. Встановлення бібліотек
Переконайтеся, що у вас встановлений Python (рекомендовано версію 3.10 або вище). Встановіть необхідні залежності:
`pip install -r requirements.txt`


### 2. Конфігурація (`config.json` та `user_data.json`)
Для повноцінної роботи потрібно створити файли конфігурації на основі прикладів:

* **ОБОВ'ЯЗКОВО**: Перейментуйте файли `config.example.json` та `user_data.example.json`

* **Gemini API**: Додайте ваш `gemini_api_key` та налаштуйте системний промпт(не обов'язково) `jarvis_system_prompt` у файлі `user_data.json`.

* **Spotify**: Вкажіть `client_id` та `client_secret` у `config.json` ([короткий відео-туторіал як їх отримати](https://youtu.be/0fhkkkRuUxw?si=U74GcDt8seGxgGlW)),  
* **Плейлисти:** HTTP-посилання на ваші улюблені плейлисти Spotify.

* **Програми та проєкти**: Пропишіть шляхи у файлі `config` до `.exe` файлів у секції `apps` та шляхи до папок у секції `projects`.

* **Telegram та Discord**: Налаштуйте імена контактів та ID серверів у `user_data.json`.

* **Озвучка**: Коротка настройка прописана в функціїї `speak_neural` в файлі `speaker.py`.

* **Відкриття головного файлу**: В файлі `open.py` можна вказати шлях до файлу `main.py`


### 3. Встановлення мовної моделі Vosk (ОБОВ'ЯЗКОВО)
Асистент використовує локальну нейромережу Vosk для розпізнавання голосу (працює без інтернету). 
1. Перейдіть на офіційний сайт Vosk: [Vosk Models](https://alphacephei.com/vosk/models)
2. Знайдіть у списку **Ukrainian** (українську) модель.
3. Завантажте архів.
4. Розпакуйте архів у головну папку цього проєкту.
5. **Важливо:** Перейменуйте розпаковану папку на `model`.
