# 🤖 Голосовий асистент "Джарвіс"

Це мій власний локальний голосовий помічник на Python, який вміє керувати комп'ютером, запускати ігри та вмикати музику голосом.

## ✨ Можливості
* Запуск і закриття програм 
* Виконування деяких системних задач (звук, мікрофон, вставка тексту і тд)
* Вводити текст який він почує
* Працювати з клавіатурою
* Керування музикою через Spotify API (готові плейлисти, перемикання треків, пошук пісень)
* Надсилання повідомлень у Telegram за допомогою гарячих клавіш
* Пошук інформації в Google та YouTube голосом

## ⚙️ Встановлення та налаштування

⚠️ **Увага:** Асистент наразі оптимізований виключно для ОС Windows 11.

### Крок 1. Завантаження проєкту
Схиліть цей репозиторій на свій комп'ютер:
`git clone https://github.com/l1br3n/ua_jarvis-by_l1br3n.git`

### Крок 2. Встановлення бібліотек
Переконайтеся, що у вас встановлений Python (рекомендовано версію 3.10 або вище). Встановіть необхідні залежності:
`pip install -r requirements.txt`

### Крок 3. Встановлення мовної моделі Vosk (ОБОВ'ЯЗКОВО)
Асистент використовує локальну нейромережу Vosk для розпізнавання голосу (працює без інтернету). 
1. Перейдіть на офіційний сайт Vosk: [Vosk Models](https://alphacephei.com/vosk/models)
2. Знайдіть у списку **Ukrainian** (українську) модель.
3. Завантажте архів.
4. Розпакуйте архів у головну папку цього проєкту.
5. **Важливо:** Перейменуйте розпаковану папку на `model`. 

### Крок 4. Налаштування конфігурації
У проєкті є два файли з прикладами налаштувань. Вам потрібно створити власні файли на їх основі:

1. **Налаштування системи (`config.json`)** Зробіть копію `config.example.json` і назвіть її `config.json`. Що туди вписати:
   * **Spotify API:** Ваші токени `client_id` та `client_secret`. (Див. [короткий відео-туторіал як їх отримати](https://youtu.be/0fhkkkRuUxw?si=U74GcDt8seGxgGlW)).
   * **Шляхи (`"apps"`):** Правильні шляхи до програм на вашому ПК (бажано, щоб файл закінчувався на `.exe` або використовуйте посилання Steam формату **steam://rungameid/айді_програми**).
   * **Процеси (`"processes"`):** Точні назви процесів для того, щоб асистент міг їх закривати (наприклад, `Telegram.exe` або `Discord.exe`).

2. **Особисті дані (`user_data.json`)**
   Зробіть копію `user_data.example.json` і назвіть її `user_data.json`. Що туди вписати:
   * **Контакти:** Юзернейми Telegram для швидкого надсилання повідомлень (без `@`).
   * **Діскорд:** ID серверів та голосових каналів для автоматичного підключення.
   * **Плейлисти:** HTTP-посилання на ваші улюблені плейлисти Spotify.
   * **Слова-синоніми (`"app_aliases"`):** Налаштуйте, як саме ви хочете називати програми голосом (наприклад, щоб на слова "майн" або "куби" бот розумів, що треба відкрити ключ `minecraft`).

## 🚀 Запуск
Коли все налаштовано, запустіть головний файл(поки працює лише в Visual Studio Code):
`python main.py`

### 💡 Вирішення проблем
* **Асистент не чує голос або вилітає з помилкою `[Errno -9999]`:** Перевірте налаштування конфіденційності Windows (Налаштування -> Конфіденційність -> Мікрофон) і переконайтеся, що терміналу дозволено доступ до мікрофона.
* **Не той голос озвучки(якщо не запускається файл з папки "sounds"):** Запустіть скрипт `check_voice.py`, щоб побачити всі встановлені голоси у вашій системі. Запам'ятайте індекс українського голосу і змініть його у файлі `actions_static.py`.
